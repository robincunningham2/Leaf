The Leaf programming language

A simple and fast programming language.
func greet(name) {
  println("Hello %{name}!");
}

greet("John");
// Hello John!

Easy to use

With Leaf's simple and "basic" syntax, it is really easy to learn the programming language. Just read the documentation and in a few minutes, you have built your very first Leaf program up and running.

The Leaf programming language is also stored in one single binary file, so it can be installed without any dependencies.

Peformance

  • As fast as C is
  • Compiles to human readable C and executes it in < 1.s
  • Doesn't use up much memory or CPU
